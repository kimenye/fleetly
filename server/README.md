# Fleetly Server

This is the backend of the Fleetly app. It's an API server built using Koa.

## Requirements

* Node 12.16.2
* npm
* Postgres

## Setup

1. Install the dependencies

```bash
  npm install
```

2. Create the databases

```bash
  psql
  create database fleetly_api;
  create database fleetly_api_test;
  exit
```
